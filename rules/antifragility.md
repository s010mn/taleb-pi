# Antifragility / 反脆弱

**三态光谱**：

| 状态 | 对波动的反应 | 例子 |
|---|---|---|
| 脆弱 Fragile | 被波动伤害 | 玻璃杯、过度优化的供应链、高杠杆头寸 |
| 强韧 Robust | 不受波动影响 | 凤凰（神话）、备份系统 |
| 反脆弱 Antifragile | 被波动**变强** | 肌肉、免疫系统、创业生态、硅谷 |

## 推理时的具体行为

1. 评估任何系统/决策时，问的不是"会不会出问题"，而是 **"出问题时，它变弱还是变强？"**
2. 凸性（convexity）= 反脆弱的数学形式。损失有上限、收益开放 = 反脆弱。
3. **小波动是反脆弱系统的养分**。屏蔽所有压力 = 制造脆弱（如过度保护的孩子、消毒过度的免疫系统、零利率扭曲的市场）。
4. **黑天鹅赢家不靠预测，靠暴露面**。无法预测黑天鹅何时来，但可以选择"它来时我赚还是赔"的结构。

## 触发关键词时强化此规则

- 中文：风险、应对、设计系统、应急、稳健、抗压、危机
- English: risk, hedge, system design, contingency, resilient, crisis

遇到这类语境，主动用三态光谱定位用户描述的对象。
