# Convexity Check / 凸性检查

## 何时使用

- 用户描述一个**会受随机性影响**的决策或系统
- 涉及**杠杆**、**期权**、**学习**、**网络效应**、**口碑**等可能非线性的结构
- 需要判断"波动是朋友还是敌人"

## 核心概念

收益函数 f(x) 对**输入波动**的反应有三种：

| 类型 | 数学 | 直觉 | 例子 |
|---|---|---|---|
| **凸 (Convex)** | f''(x) > 0 | 平均输出 > 输出平均；波动有利 | 期权、创业、写作传播、肌肉训练 |
| **线性 (Linear)** | f''(x) = 0 | 波动无关紧要 | 固定工资 |
| **凹 (Concave)** | f''(x) < 0 | 平均输出 < 输出平均；波动有害 | 高杠杆头寸、过度优化系统、易碎玻璃 |

**詹森不等式**：对于凸函数，E[f(X)] ≥ f(E[X])；对于凹函数反之。
**翻译**：在凸结构里，**承担可控波动比追求平均稳定更赚**；在凹结构里反之。

## 分析步骤

### 1. 找到收益函数的形状
画出"输入压力 → 收益"的关系：
- 输入翻倍时，收益翻多少倍？
- 输入跌一半时，损失多少？
- 在极端输入下（10x、100x、0.01x），函数怎么走？

### 2. 看二阶导数（曲率）
- **下行有界 + 上行开放** = 凸
- **上行有界 + 下行开放** = 凹（**危险**）
- **两端都开放** = 看具体形状，通常是骗局或赌博

### 3. 判定建议
- 凸结构 → **增加暴露**，让波动滋养
- 凹结构 → **减少暴露 / 添加保险 / 退出**
- 线性结构 → 波动不重要，专注期望值

## 隐藏的凹性（最危险）

很多看似平稳的事物**藏着凹性**：

- **零库存供应链**：99% 的时间高效，1% 的极端事件全线崩溃
- **优化到极致的代码**：性能 +5%，但任何小输入异常都崩
- **高杠杆稳定回报基金**：年年赚 8%，第 10 年归零
- **完美健康饮食**：错过任何一次饥荒压力，反而免疫脆弱
- **过度专业化职业**：技能极强但行业被颠覆时无法转移

塔勒布的判别问题：**"这个东西在过去 X 年没出大问题，是因为它强韧，还是因为它没遇到考验？"**

## 隐藏的凸性（最被低估）

- **写作 / 公开内容**：99% 的文章无人看，1% 的爆文带来巨大杠杆
- **结识新人**：99% 的认识没用，1% 改变命运
- **小额风险投资 / 长尾期权**：单笔归零，组合超额回报
- **学习外行知识**：99% 用不上，1% 跨界破局

## 输出格式

```
对象：           [用户描述的决策/系统]
收益函数形状：   [凸 / 线性 / 凹 / 混合]
二阶曲率证据：   [输入翻倍/减半时的具体反应]
隐藏凸/凹性：    [是否有时间维度上的反转]
建议：          [增加暴露 / 减少暴露 / 加对冲 / 退出]
```
