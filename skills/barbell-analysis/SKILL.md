# Barbell Analysis / 杠铃策略

## 何时使用

- 用户面临**资源配置**决策（资金、时间、注意力、职业方向）
- 涉及**不确定性较高**的领域（投资、创业、学习选择）
- 用户在"中庸方案"和"激进方案"之间纠结

## 核心方法

杠铃策略 = **两端极端 + 回避中间**：

```
安全端 (85-90%)  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━  激进端 (10-15%)
  下行有限                                        上行开放
  现金/国债/低杠杆                                 创业/期权/小赌注
  本职工作                                          副业/写作/投资
  低风险方向                                        非对称机会
                  ╳ 中间地带是陷阱 ╳
            (高估值蓝筹、中等杠杆、稳定无聊)
```

## 分析步骤

1. **画出收益分布**：用户描述的方案是凸还是凹？最坏情况？最好情况？
2. **识别中间陷阱**：是否在追求"看起来稳但暗含尾部风险"的中间方案？
3. **重构为杠铃**：能否把同样的预算拆成 90% 极保守 + 10% 极激进？
4. **检查激进端**：激进端的下行是否真的有限？（不是"我感觉不会归零"，而是"机制上不可能归零"）
5. **检查保守端**：保守端是否真的保守？（国债在通胀里也会赔；现金在恶性通胀里就是手纸）

## 输出格式

回答用户时按这个结构：

```
你的当前配置：[描述用户方案在杠铃光谱上的位置]
风险结构：    [识别凹凸性、最坏情况]
重构建议：    [给出杠铃版本]
保守端检查：  [保守端是否真保守]
激进端检查：  [激进端下行是否真有限]
```

## 经典例子

- **塔勒布本人的投资**：85-90% 国债 + 10-15% 深度虚值期权
- **职业杠铃**：稳定本职 + 副业写作/创业（而不是辞职 all-in 创业，也不是只做无聊本职）
- **学习杠铃**：经典/林迪知识 + 前沿小赌注（而不是中等热门技术堆栈）
- **饮食杠铃**：长期禁食 + 偶尔暴饮暴食（而不是稳定均衡饮食）

## 反模式

- "我们做一个分散投资组合" → 这是均衡，不是杠铃
- "我做一个有适度风险的方案" → 适度风险通常意味着尾部暴露被低估
- "我两边都试试" → 试试是对的，但比例对吗？应该 90/10，不是 50/50
